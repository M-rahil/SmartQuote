# SmartQuote

Full-stack freelance pricing intelligence platform built with React, Node.js, Express.js, MongoDB, and Tailwind CSS.

## Features

- JWT Authentication
- Real-time quote generation
- Multi-factor pricing engine
- PDF proposal generation
- Analytics dashboard
- Quote history management
- MongoDB data persistence

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Recharts

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Pricing Engine

Factors used:

- Skill
- Experience Level
- Country
- Project Complexity
- Urgency
- Profit Margin
- Overhead Costs

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Deployment

- Frontend: Vercel
- Backend: Render
